
# Mordred Engine

Mordred Engine is a proprietary, Entity Component System Based, engine being developed by Normal Sink Studios using the FLECS ECS Library

# Requirements
c++ 17 || Cmake || OpenGL
## Installation

Installing the Mordred Engine 

```bash
  git clone https://github.com/Normal-Sink-Studio/Mordred-Engine.git
  cd Mordred-Engine
```

## Building
To compile the Mordred Engine use the following commands
```bash
mkdir build
cmake -B ./build -S ./src
make
```
